# HMPI Galaxy Showcase Frontend

This folder contains a modern, pure HTML/CSS/JavaScript UI for the HMPI Galaxy project. It includes:

- Hero section
- Automated HMPI Calculator
- AI Predictive Forecast
- Interactive GIS Dashboard
- Global Statistics
- Manual HMPI Diagnostic Tool
- About & Tech Stack
- Contact & Feedback

## How to Run

Open `index.html` in your browser. All logic is in `app.js` and styles in `style.css`.
